# repoCreateTest

A simple repository demonstrating a random joke script.

## Run

```bash
python3 hello_world.py
python3 "2+2=10.py"
python3 mountain_joke.py
python3 toddler_stories.py
```

## Usage

```python
from add_two_numbers import add
print(add(1, 2))  # 3
```

## Tests

Run the unit tests with:

```bash
python -m unittest -q
```
